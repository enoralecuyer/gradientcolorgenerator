# Gradient Color Generator

Create your own color gradient and save its values for your own project, for example:

```
rgba(0, 0, 0, 0) linear-gradient(to right, rgb(71, 131, 184), rgb(255, 0, 0)) repeat scroll 0% 0%;
```
